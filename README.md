# Mon Premier Projet Web

Un projet simple pour apprendre à déployer sur **GitHub** et **Netlify**.

## Structure des fichiers

```
mon-projet/
├── index.html        ← La page principale
├── style.css         ← Les styles (couleurs, mise en page)
├── README.md         ← Ce fichier
└── images/
    ├── montagne.svg  ← Illustration montagne
    └── ocean.svg     ← Illustration océan
```

## Comment déployer

1. **GitHub** : Créer un dépôt, puis glisser ces fichiers ou utiliser `git push`
2. **Netlify** : Connecter le dépôt GitHub → déploiement automatique !
